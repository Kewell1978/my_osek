README !

FreeOSEK on STM32F1 ARM Cortex-M3 micro-controller

EmBitz IDE                     : EmBitz 0.42

UART Debug Connection:

PA9  (UART1_TX) <--> RS232 RX
PA10 (UART1_RX) <--> RS232 TX

Normally OSEK oil generator is disabled to enable :

From Menu -> Project -> Build options -> Pre/post build steps -> change drop box to enable.
(This step is need only one time right after changing config.oil file)
